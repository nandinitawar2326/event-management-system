# 🎉 Event Management System

A full-stack Event Management System built using **React, Node.js, Express.js, and MySQL**. The application allows users to register, log in, browse events, book events, and manage their bookings. It also includes an admin panel to view users and bookings.

---

## 🚀 Features

### 👤 User
- User Registration & Login
- Secure password encryption using bcrypt
- Browse available events
- Book events
- View My Bookings
- Cancel bookings

### 👨‍💼 Admin
- View all registered users
- View all event bookings
- Manage event records

## ⚙ Installation

### Clone the repository

```bash
git clone https://github.com/nandinitawar2326/event-management-system.git
```

### Navigate to the project

```bash
cd event-management-system
```

---

## Install Frontend

```bash
cd client
npm install
npm start
```

Runs on:

```
http://localhost:3000
```

---

## Install Backend

```bash
cd server
npm install
npm start
```

Runs on:

```
http://localhost:5000
```

---

## 🔐 Environment Variables

Create a `.env` file inside the **server** folder.

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=event_db
PORT=5000
```
